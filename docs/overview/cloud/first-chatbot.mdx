# Introduction

The botpress cloud is out. This version is a little bit different from the open source version. The beta version will be focusing on the the deployment part. In this version we have build a conveniant way to deploy a chatbot. The Frontend is the same but with extra feature. You can use the atom version or the latest docker release. 

This new product is better in lot of way. It solve the time-to-production, in one click, you have a deployed chatbot that can work on various platform. It also solve, the collaboration between conversation designer. It come with all the tool for post-production (manage and improvement). Enought of talking, let's jump into the lesson. 


## Create an account on botpress cloud

1. Go on the botpress [cloud](https://app.botpress.dev) 

2. Create an account

## Download the binary

You can download the binary or use the latest version of the docker.
```
docker run -p 3000:3000 --rm -it ghcr.io/botpress/botpress/server:next
```

## Create a chatbot in the studio

In the studio,  

1. `Create Bot` > `New Bot` > `Enable Botpress Cloud`
2. Copy the keys from the botpress cloud.
3. Create a node text.
4. Train your chatbot

## In the cloud platform, enable telegram

1. Open the botfather on [telegram](https://t.me/botfather)
2. Start the conversation
3. Type `/newbot` and follow the instruction.

## Deploy your chatbot

1. Get the HTTP API from the botfather.
2. Copy the keys, in the bot token 
3. Deploy your chatbot
